FOSS - What is it?

Definition

Free and open-source software (FOSS) is a term used to refer to groups of software consisting of both free software and open-source software where anyone is freely licensed to use, study, modify and change the software in any way and the source code is openly shared so that people are encouraged to voluntarily improve the design of the software.

History

Frustrated with the shift towards proprietary software, in 1983 Richard Stallman announced the GNU project to develop a free and open-source operating system. The Free Software Foundation (FSF) was founded in 1985 to oversee the project, and Stallman published the GNU Manifesto the same year. The manifesto introduced the Free Software Definition and "copyleft" ideas.

In 1991, Linus Torvalds released the Linux kernel as freely modifiable source code. Initially, it wasn't released under a Free software or Open-source software license. However, with version 0.12 in February 1992, he relicensed the project under the GNU General Public License.

In 1997, Eric Raymond published The Cathedral and the Bazaar, a reflective analysis of the hacker community and Free software principles. The paper received significant attention and was one factor in motivating Netscape Communications Corporation to release their popular Netscape Communicator Internet suite as Free software. This code is today better known as Mozilla Firefox and Thunderbird
